ゼロトラスト成熟度評価CLI — 5分でセットアップ完了

STEP 1

動作環境の確認

ZeroVis JP は Python 3.9 以上が必要です。

python --version
# Python 3.9.x 以上であれば OK
STEP 2

リポジトリのクローンとインストール

git clone https://github.com/yorosiku008/zerovis-cli.git
cd zerovis-cli
pip install -r requirements.txt
依存パッケージは rich（ターミナル描画）のみです。外部APIへの接続は不要です。
STEP 3

デモモードで動作確認

組織種別を指定してゼロトラスト成熟度をシミュレーションできます。

# 一般組織でデモ
python main.py --demo

# 自治体向けデモ
python main.py --demo --org-type municipal

# 製造業向けデモ
python main.py --demo --org-type manufacturing
4軸スコアバー、成熟度グレード、改善項目が表示されれば成功です。

STEP 6

Markdownレポートの出力

python main.py --demo --org-type municipal --output-md
# zerovis_report_YYYYMMDD.md が保存されます
レポートには成熟度スコア、軸別評価、改善項目一覧が含まれます。IT導入補助金申請書の参考資料として活用できます。
